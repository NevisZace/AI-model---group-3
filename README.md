The dataset used came from https://www.kaggle.com/datasets/janiobachmann/bank-marketing-dataset/data

The project investigates a binary classification problem using the Bank Marketing dataset, where the objective is to predict whether a customer will subscribe to a term deposit (deposit = yes/no) based on demographic and behavioural features.

using different supervised and unsupervised models imported from scikit-learn

main.py – Initial exploratory data analysis (EDA)

baseline_majority.py – Majority class baseline model

decisiontree.py – Decision tree model with CV, overfitting analysis, and GridSearchCV

random_forest.py – Random forest classifier

linear_regression.py – Linear regression model

k-means_clustering.py – Unsupervised clustering analysis

pca.py – Principal component analysis

bank.csv – Dataset used for analysis
